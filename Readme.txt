== Description ==
新淘客Wordpress插件支持主流电子商务平台(淘宝，拍拍，其他B2C商城)推广赚取佣金，支持会员购物分享，购买返现，推广返现，分享返现，站长可个性化装修设计所有页面。。更多详情:** http://plugin.xintaonet.com

= V1.1.2详细描述 =

一. 支持淘宝(无需淘宝客API权限),天猫,聚划算,拍拍,其他主流B2C商城(京东,凡客,一号店...)的商品,店铺推广

二. 支持会员购物分享,会员可分享来自淘宝,天猫,聚划算,拍拍等商城的商品

三. 支持会员购买返现,推广返现,分享返现,站长可自由开启或关闭,支持配置各种返现比例

四. 在新淘客平台中站长可个性化配置站点皮肤主题,任意装修设计所有站点页面

五. 在新淘客平台中站长可配置自动分享，系统每天自动为您分享优质商品

六. 提供商品采集插件，目前支持蘑菇街，美丽说采集，稍后支持更多网站采集

七. 提供淘宝客API接口，避免初级阶段无法申请到淘宝客API接口

= 注意事项 =

1、安装完新淘客WordPress插件后,请及时进入新淘客平台(http://plugin.xintaonet.com)添加您的网站

2、所有个性化装修功能均在新淘客平台完成,插件本身并不提供个性化装修设计功能

== Installation ==

1.下载 新淘客WordPress 插件，上传wp-xintaoke目录及其文件到 "/wp-content/plugins/" 插件目录，在后台管理中激活插件，根据提示完成所有配置功能.
2.注册，登陆新淘客平台(http://plugin.xintaonet.com)添加您的网站，即可完成站点初始化.
